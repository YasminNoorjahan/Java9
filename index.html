4. Stream API Improvement

  • takeWhile and dropWhile api:

    • Unlike Filter, takeWhile: Prints the elements until the first element that does not satisfy the condition .dropWhile: Skips the elements until the first element that does not satisfy the condition (even numbers), and prints the rest.

                
                // The Below prints 20 only. It is the first element satisfy the condition so the preocess stops there itself
List<Integer> immutableList = List.of(20, 13, 45, 89, 67, 34, 90);
System.out.println("Even Numbers");
immutableList.stream().takeWhile(number -> number % 2 == 0).collect(Collectors.toList()).forEach(System.out::println);
 
 // The Below prints all elements except 20. 
System.out.println("Odd Numbers");
immutableList.stream().dropWhile(number -> number % 2 == 0).collect(Collectors.toList()).forEach(System.out::println);
